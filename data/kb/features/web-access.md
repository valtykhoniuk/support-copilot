# Web Access and Device Limits

FoxSchool is a web application optimized for modern browsers. There is no native mobile app as of March 2026; the mobile web experience is fully supported.

## Supported browsers

| Browser | Minimum version |
|---------|-----------------|
| Chrome | 110+ |
| Firefox | 110+ |
| Safari | 16+ |
| Edge | 110+ |

Internet Explorer is not supported.

## Device session limits

| Limit | Value |
|-------|-------|
| Simultaneous active sessions | 3 |
| Session timeout (inactive) | 24 hours |
| Video quality (auto) | Up to 1080p on connections faster than 5 Mbps |

If you exceed 3 devices, the oldest session is logged out automatically with an email notification.

## Mobile web tips

- Add to home screen for quick access (Chrome: Menu → Add to Home screen)
- Download lessons for offline viewing: **not available** on any plan (online only)
- Use headphones for speaking exercises—microphone access is required

## System requirements

| Component | Requirement |
|-----------|-------------|
| Internet speed | 2 Mbps minimum, 5 Mbps recommended for HD video |
| Microphone | Required for speaking exercises (optional to skip) |
| Screen | 320px width minimum |

## Switching devices mid-lesson

Progress saves automatically after each exercise. If you switch devices:

1. Close the lesson on the first device (or wait 30 seconds for auto-save)
2. Open the same lesson on the new device
3. You resume at the last completed exercise
