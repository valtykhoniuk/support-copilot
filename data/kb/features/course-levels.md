# CEFR Levels and Course Structure

FoxSchool organizes all content using the Common European Framework of Reference for Languages (CEFR). Your subscription plan determines which levels you can access.

## Level overview

| CEFR Level | Description | Typical study time | FoxSchool modules |
|------------|-------------|-------------------|-------------------|
| **A1** | Beginner — basic phrases, introductions | 60–80 hours | 12 modules |
| **A2** | Elementary — routine tasks, simple exchanges | 80–100 hours | 12 modules |
| **B1** | Intermediate — travel, work, familiar topics | 100–120 hours | 13 modules |
| **B2** | Upper intermediate — complex texts, fluent interaction | 120–150 hours | 13 modules |
| **C1** | Advanced — demanding texts, flexible language use | 150–180 hours | 11 modules |
| **C2** | Mastery — near-native nuance and precision | 180–200 hours | 11 modules |

## Plan-to-level mapping

| Plan | Monthly price | Unlocked levels |
|------|---------------|-----------------|
| Beginner | $20 | A1, A2 |
| Intermediate | $30 | B1, B2 |
| Advanced | $40 | C1, C2 |

You cannot access B1 content on the Beginner plan, even if you pass the placement test at B1. You must upgrade to Intermediate ($30/month).

## Module structure

Each module contains 4 lessons. A lesson includes:

1. Video instruction (6–12 minutes)
2. Vocabulary set (20–30 words)
3. Grammar exercises (10 questions)
4. Module quiz (15 questions, 70% pass required)

## Placement test

The placement test is free and takes 15 minutes. It contains 40 questions across reading, listening, and grammar. Results recommend a starting module:

| Score range | Recommended start |
|-------------|-------------------|
| 0–10 | A1 Module 1 |
| 11–20 | A1 Module 6 |
| 21–28 | A2 Module 1 |
| 29–34 | B1 Module 1 |
| 35–38 | B2 Module 1 |
| 39–40 | C1 Module 1 |

Placement results do not change your plan—you still need the matching subscription to access higher levels.

## Certificates

Optional certificates of completion are available for $15 per level after passing the level final exam (80% required). Certificates are PDF downloads, not accredited diplomas.
