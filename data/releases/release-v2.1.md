# FoxSchool Release Notes — v2.1

**Release date:** March 1, 2026

---

## New Features

### Speaking practice beta (A1–A2)
Interactive speaking prompts now available in all A1 and A2 lessons. Uses browser microphone with real-time waveform feedback. Optional—skipping does not block progress.

### Progress CSV export
Download your full learning history from Settings → Progress → Download report. Includes module names, completion dates, quiz scores, and total study minutes.

### Mentor session packs
Purchase 5 mentor sessions for $90 (save $10). Packs expire 90 days after purchase. Available from Dashboard → Book a mentor.

## Improvements

### Video player performance
Reduced initial load time by 40% on connections under 5 Mbps. Auto-quality algorithm now drops to 360p faster to prevent buffering.

### Quiz retry cooldown
Failed quiz retry cooldown reduced from 8 hours to **4 hours**.

## Bug Fixes

- Fixed dashboard streak not incrementing when completing lessons near midnight UTC
- Fixed PayPal renewal failing silently for accounts created before January 2026
- Fixed PDF worksheet download returning 404 for A2 Module 8
- Fixed placement test audio not playing on Safari 17.2
- Fixed mentor booking calendar showing wrong timezone for users in UTC+5:30

## Known Issues

- Speaking exercises may fail on Safari versions below 17.4—use Chrome as a workaround. Fix planned for v2.2.
